# Hindsight VS Code Extension

All notable changes to the Hindsight MCP VS Code extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.5] - 2026-01-19

### Added

- Initial release of Hindsight MCP VS Code extension
- Automatic binary download and management for all supported platforms
- MCP server definition provider for VS Code integration
- Configuration options for database path and binary location
- Commands: Download Binary, Show Version
- Output channel for logging and debugging
- Support for Linux (x64, ARM64), macOS (Intel, Apple Silicon), and Windows (x64)
